## JavaScript 简介
一个完整的JavaScript 实现应该由下面三个部分组成：
<ul>
<li>核心（ECMAScript）</li>
<li> 文本对象模型（DOM）</li>
<li> 浏览器对象模型（BOM）</li>
</ul>

### 小结
  JavaScript是一种转为与网页交互而设计的脚本语言，由以下三个部分组成：
  <ul>
  <li>ECMAScript,提供核心语言功能</li>
  <li>文本对象模型（DOM），提供访问和操作网页内容的方法和接口</li>
  <li>浏览器对象模型（BOM），提供与浏览器交互的方法和接口</li>
  </ul>
